# Ejercicios: Backtracking

1. **Subset Sum.** Dado un conjunto de números enteros y un valor
   objetivo `k`, determinar si existe algún subconjunto cuya suma
   sea exactamente `k`. Implementar la solución con backtracking.
   → `01-subset-sum/`

2. **Cambio de Monedas.** Dado un monto y un conjunto de denominaciones,
   encontrar todas las combinaciones posibles de monedas que suman
   exactamente el monto. Usar backtracking para generar todas las
   soluciones.
   → `02-cambio/`

3. **Laberinto.** Dada una matriz que representa un laberinto
   (0 = paso libre, 1 = obstáculo), encontrar un camino desde la
   posición `(0,0)` hasta `(N-1, N-1)` usando backtracking.
   Solo se permite moverse hacia abajo y hacia la derecha.
   → `03-laberinto/`
